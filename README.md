# Web Client Config

## Instalação

Inicialmente inclua em seu site:
``` HTML
<script src="tcc.js"></script>
```
E construa um objeto do tipo TCC

``` javascript
<script type="text/javascript">
	$( document ).ready(function() {
	    var tcc = new TCC(
	      param1: value,
	      param2: value,
	      ...
	    );
	});
</script>
```
